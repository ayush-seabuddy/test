# SeaBuddy

SeaBuddy is a React Native mobile application designed to deliver a smooth and engaging experience on both Android and iOS platforms.

---

## Getting Started

### Installation
```bash
git clone https://github.com/Seekware-Private-Limited/seabuddy-app-prod.git
cd seabuddy
npm install
````

### Run the App

```bash
npm run android   # for Android
npm run ios       # for iOS
npm start         # start Metro Bundler
```
---

## Common Commands

| Command           | Description         |
| ----------------- | ------------------- |
| `npm start`       | Start Metro Bundler |
| `npm run android` | Run on Android      |
| `npm run ios`     | Run on iOS          |
| `npm run test`    | Run test cases      |
